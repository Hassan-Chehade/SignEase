=============================================
  PYTHON SERVICE  (D:\signease\python-service\)
=============================================

Framework:  FastAPI + uvicorn
Python:     3.13.7

Files:
  main.py             — FastAPI app (see logic below)
  requirements.txt    — all deps (mediapipe, fastapi, transformers, torch, etc.)
  .env                — HOST=0.0.0.0, PORT=8001
  hand_landmarker.task — Downloaded MediaPipe hand landmarker model (already on disk)

Pre-trained model:
  HuggingFace: prithivMLmods/Alphabet-Sign-Language-Detection
  Architecture: SiglipForImageClassification (fine-tuned from google/siglip2-base-patch16-224)
  Classes: A-Z (26 ASL alphabet letters)
  Accuracy: 99.96%
  Size: ~400MB, downloaded on first run, cached by HuggingFace hub
  NO training required — fully pre-trained

main.py logic:
  - lifespan: loads HuggingFace pipeline on startup (top_k=3)
  - MediaPipe Tasks HandLandmarker: crops hand ROI with 15% padding
    - If no hand found → returns {detected: false}
    - Falls back to full frame if MediaPipe errors
  - POST /detect: decode base64 → crop hand → classify → return top sign + alternatives
  - GET  /health: returns {status, model_loaded, mediapipe}

MediaPipe version: 0.10.35 (supports Python 3.13)
  - Uses Tasks API (mp_vision.HandLandmarker), NOT the old Solutions API
  - hand_landmarker.task file downloaded from Google Storage

Torch version installed: 2.11.0+cpu

Key fix applied: mediapipe 0.10.35 uses min_hand_presence_confidence
  (not min_hand_presence_score which was the old param name)

=============================================
  KNOWN NOTES
=============================================

- Model gives low confidence on blank/test images (expected)
  Real hand signs will get high confidence (model is 99.96% accurate on ASL dataset)

- The 0.6 confidence threshold in App.jsx prevents junk detections from
  being appended to the text builder

- Detection debounce: same letter won't append again for 1500ms,
  forcing user to briefly move hand away between repeated letters

- SQLite DB is at backend/database/database.sqlite (no separate DB server needed)

- If port 8001 already in use: kill with
  Get-NetTCPConnection -LocalPort 8001 | % { Stop-Process -Id $_.OwningProcess -Force }

- TF_ENABLE_ONEDNN_OPTS=0 can be set to suppress TensorFlow info logs from mediapipe
